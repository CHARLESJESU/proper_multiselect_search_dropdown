## 1.0.0
- Initial release of `proper_multiselect_search_dropdown`.
- Added multi-select dropdown widget with search functionality.
- Provided example app demonstrating how to use the package.

## [1.0.1] - 2025-12-06

### Added
- Initial release
- Multi-select dropdown with search functionality
- Real-time search with text highlighting
- Support for title-only and title+subtitle items
- Two layout modes: inline and two-line
- Highly customizable styling options
- Keyboard-aware layout adjustment
- Customizable icons, colors, and text styles
- Checkbox position customization (left/right)
- Search box customization
- Smart search sorting (exact → starts-with → contains)

### Features
- Material Design compliant
- Null-safe
- Supports Flutter 3.0. 0+
- Responsive and adaptive to screen size
