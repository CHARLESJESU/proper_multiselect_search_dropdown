## [1.0.3]

### Added
- ProperSingleSelectDropdown widget for single-selection use cases
- enableSearch parameter for both Multi-Select and Single-Select widgets
- autoCloseOnSelect option for single-select dropdown (default: true)
- showClearButton option for single-select dropdown (default: true)
- Conditional search functionality (enable/disable with enableSearch parameter)

### Features
- Single-select with visual checkmark indicator
- Optional search functionality for better performance
- Smart text highlighting only when search is enabled
- Keyboard-aware with conditional auto-focus
- Material Design compliant
- Null-safe
- Supports Flutter 3.0.0+

## [1.0.2]

### Added
- Initial release
- Multi-select dropdown with search functionality
- Real-time search with text highlighting
- Support for title-only and title+subtitle items
- Two layout modes: inline and two-line
- Highly customizable styling options
- Keyboard-aware layout adjustment
- Customizable icons, colors, and text styles
- Checkbox position customization (left/right)
- Search box customization
- Smart search sorting (exact → starts-with → contains)

### Features
- Material Design compliant
- Null-safe
- Supports Flutter 3.0.0+
- Responsive and adaptive to screen size
